Machine Learning Formulas

Linear Regression Equation

Y = mX + c

Y = Predicted Value

X = Input Variable

m = Slope

c = Intercept

Mean Squared Error (MSE)

MSE = (1/n) × Σ(y − ŷ)²

Mean Absolute Error (MAE)

MAE = (1/n) × Σ|y − ŷ|

Root Mean Squared Error (RMSE)

RMSE = √MSE

R² Score

R² = 1 − (SSR / SST)
